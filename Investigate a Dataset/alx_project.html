Introduction¶

Dataset Description¶

This dataset contains the medical info of 62,299 patients and 110,527 appointments in various cities of Brazil. The dataset is focused on whether or not patients always honour the appointments set with the hospital and the various variables that affect it.

The dataset contained the following columns:</br> PatientID: An ID unique to every individual that comes to a hospital.</br> AppointmentID: An ID unique to every appointment in the hospital.</br> Gender: M or F </br> ScheduledDay: The day the appoinment was scheduled.</br> Appointment Day: The day of the appointment.</br> Age: How old the patient is.</br> Neighbourhood: Location of the hospital.</br> Scholarship: Financial aid to poor Brazilian families</br> Hypertension: 0 or 1, stands for False or True.</br> Diabetes: 0 or 1, stands for False or True.</br> Alcoholism: 0 or 1, stands for False or True.</br> Handcap: 0 or 1, stands for False or True.</br> SMS_received: 0 or 1, stands for False or True.</br> No_show: No or Yes, No stands for if the appointment was honoured and vice versa</br>

A function for plotting a barchart is defined below. It has 7 parameters

  • x: A list of numbers that contains the length of the x-axis
  • y: A list of the first bars in each pair
  • z: A list of the second bars in each pair
  • labels: The labels of the plots, separated by a whitespace
  • x_label: Label of the x-axis
  • y_label: Label of the y-axis
  • title: Title of the plot
In [30]:
def plotbar(x,y,z,labels,x_label,y_label,title):
    x_axis=np.arange(len(x))
    plt.bar(x_axis - 0.2, y, 0.4, label = labels.split(' ')[0])
    plt.bar(x_axis + 0.2, z, 0.4, label = labels.split(' ')[1])
    
    plt.xticks(x_axis, x)
    plt.xlabel(x_label)
    plt.ylabel(y_label)
    plt.title(title)
    plt.legend()

Limitations¶

  • It would have been better if the deatils of the number of SMS received was also included, not 0 and 1.
  • The fact that the Handcap column had many values that are not 0 or 1 was also a limitation.

Conclusions¶

  1. About 80% of appointments were fulfilled while about 20% was not fulfilled.
  2. Out of 62298 patients, there were 22,253 males and 40,0045 females.
  3. Male and female patients have almost the same proportion of appointments.
  4. The youths (18 - 35years) are the worst at keeping to appointments.
  5. The SMS sending system is not effective.
